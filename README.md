# Feed The Noik!

## Game Engine

This game is developed in the custom engine, made by myself (ten-trillion).

This engine will be optimized to render rendering sequences very quickly with SDL2.

## Pre-Requirements

- xmake
- git
- c99 (C 99 ISO Standart)

To setup this project, you need [XMake](https://xmake.io/). The build system for this project.
